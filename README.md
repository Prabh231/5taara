# 5 Taara Punjabi Restaurant

## Project Purpose

5 Taara Punjabi Restaurant is a fictional Punjabi restaurant website created for this Project. The purpose of this project is to design a professional and responsive website that showcases Punjabi food products and allows users to explore menu items, product details, and restaurant information.

## Technologies Used
- HTML5
- CSS3
- Google Fonts
- GitHub 
## Project Pages
The website includes the following pages:
- Homepage (`index.html`)
- About Page (`about.html`)
- Shop/Menu Page (`shop.html`)
- Contact Page (`contact.html`)
- Amritsari Kulcha Product Page
- Chole Bhature Product Page
- Gol Gappa Product Page
## Features
- Semantic HTML
- Responsive design for small screens
- proper color sceme
- Custom CSS styling
- Product cards and detailed product pages
- Contact form
- Navigation between all pages
- Food images and visual sections
## Setup Instructions
To run this project locally:
1. Download or clone the GitHub repository.
2. Open the project folder.
3. Open `index.html` in a web browser.
4. Navigate through the website using the navigation menu.
or
1. open vis github page or live server link
No additional software or installation is required.
